# Итоговое задание по курсу «Базы данных»

**Вариант:** 1 — Учебное заведение (студенты, преподаватели, предметы, оценки)
**СУБД:** MySQL 8.0
**Среда выполнения:** Ubuntu (CLI, mysql)

---

## 1. Описание проекта

В репозитории представлено итоговое задание по курсу «Базы данных».
В рамках задания спроектирована и реализована реляционная база данных для учёта учебного процесса в образовательном учреждении.

База данных позволяет:
- хранить информацию о студентах и учебных группах;
- учитывать преподавателей и учебные предметы;
- фиксировать факты преподавания предметов в конкретных семестрах;
- хранить оценки студентов;
- выполнять аналитические и управленческие SQL-запросы.

---

## 2. Содержимое репозитория

В репозитории содержатся следующие файлы:

- `ddl.sql` — SQL-скрипт создания базы данных, таблиц, первичных и внешних ключей, ограничений;
- `seed.sql` — тестовые данные, обеспечивающие работу всех запросов;
- `queries.sql` — SQL-запросы согласно требованиям задания (аналитические запросы и CRUD);
- `report.md` — отчёт с описанием структуры базы данных и реализованных решений;
- `README.md` — инструкция по проверке проекта.

---

## 3. Требования для запуска

Для проверки проекта требуется:
- установленный MySQL версии 8.0;
- доступ к CLI-клиенту `mysql`.

Использование графических инструментов (например, MySQL Workbench) не является обязательным.

---

## 4. Инструкция по запуску

Все команды выполняются из корневой директории репозитория.

Шаг 1. Создание структуры базы данных
```
mysql -u root -p < ddl.sql
```

Шаг 2. Загрузка тестовых данных
```
mysql -u root -p education_db < seed.sql
```

Шаг 3. Выполнение SQL-запросов 
```
mysql -u root -p education_db < queries.sql
```

После выполнения команды будут выведены результаты аналитических SQL-запросов.

---

## 5. Проверка корректности загрузки данных (опционально)

Для быстрой проверки можно выполнить следующие SQL-команды:
```
SHOW TABLES;
SELECT COUNT(*) FROM students;
SELECT COUNT(*) FROM grades;
```

---

## 6. Примечания

- Проект выполнен в среде Ubuntu с использованием командной строки.
- Все SQL-скрипты воспроизводимы на любой системе с установленным MySQL 8.0.
- База данных не требует дополнительной настройки окружения.
